== Description ==

This WordPress plugin adds the Lightbox feature to image links and to the new 
WordPress 2.5 gallery feature too. It uses the jQuery Lightbox Plugin by Benjamin 
Arthur Lupton.

---------------------------------------------------------------------------------------

== Installation ==

   1. Download the file using the link below and unzip it.
   2. Upload the folder 'wp_gallery_lightbox' to your 'wp-content/plugins' folder
   3. Activate the plugin from the plugins page
